Zappy

Description

Zappy is a real-time multiplayer strategy simulation developed as part of the Epitech curriculum. It simulates a competitive environment where multiple AI-controlled players interact on a server-controlled map.

The project is structured around a client-server architecture written in C. The server manages the game world, including the map, resources, and all player actions. Only AI clients connect to the server, competing to fulfill victory conditions through cooperation and strategy. A separate graphical client can also connect to observe the game in real time.

Features

  • Client connection management using poll
  • Artificial intelligence playing automatically
  • Real-time synchronization of AI actions
  • Server managing the map, resources, and game state
  • Network commands to control and communicate between AIs
  • Graphical client to observe the game in real time

Technologies

  • Server in C
  • GUI in C++
  • AI in Python

Installation

Compile the server and the GUI :

make

Usage

  1. Start the server :
USAGE: ./zappy_server -p port -x width -y height -n name1 name2 ... -c clientsNb -f freq
port is the port number
width is the width of the world
height is the height of the world
nameX is the name of the team X
clientsNb is the number of authorized clients per team
freq is the reciprocal of time unit for execution of actions
  1. Start the GUI :
USAGE: ./zappy_gui -p port -h machine
port is the port number
machine is the hostname of the server
  1. Start the AIs :
USAGE: ./zappy_ai -p port -n name -h machine
port is the port number
name is the name of the team
machine is the hostname of the server

Project structure

  • server/ : server source code
  • gui/ : gui source code
  • ai : AI source code
